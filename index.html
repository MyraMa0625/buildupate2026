Microsoft IQ — the context layer

A unified intelligence layer that grounds agents in both enterprise and world knowledge, available across GitHub Copilot, Microsoft Foundry and Copilot Studio.

Work IQ GA · APIs Jun 16

Workplace intelligence capturing how work happens across Microsoft 365 — people, emails, documents, meetings and how they connect.

Web IQ New Limited access

AI-first, MCP-native web grounding stack returning relevant passages at ~2.5x the speed of the next best alternative.

Foundry IQ GA

Ties knowledge together and enables retrieval planning across enterprise knowledge and the live web.

Fabric IQ New

A shared semantic foundation (ontology) over structured business data for agents to reason on.

Microsoft Foundry — build & run agents

The production layer for deploying, improving and trusting agents at scale.

Microsoft Agent Framework 1.0 GA

Production-grade orchestration for .NET and Python with Agent-to-Agent (A2A) protocol so agents on different runtimes coordinate.

Hosted Agents GA by end of June

Hypervisor-isolated sandboxing, per-agent Entra ID, source-code deploy via azd, built-in content safety and Voice Live.

11,000+ model catalogue GA

Includes OpenAI GPT-5.5, Anthropic Claude Opus 4.8 (preview), Fireworks AI, Mistral, DeepSeek and more — model-diverse by design.

Foundry Toolkit for VS Code GA

Bring agent build, test and deployment workflows directly into the editor.

Copilot, Scout & Agent 365

Agents reach users — and IT gets the controls to govern them.

Microsoft Scout New product

An always-on personal "Autopilot" agent for work with its own identity, built on OpenClaw. Desktop app for Windows 11+ and macOS, available through the Frontier preview.

A new look for Copilot New

Copilot becomes a superapp uniting chat, Cowork and code in a single experience.

Agent 365 SDK GA

The governance umbrella for agents — framework-agnostic SDK, Local Agents discovery (Claude Code, Copilot CLI, OpenClaw) in preview.

Copilot Credits New

The consumption meter for agent work — pay-as-you-go ($0.01/credit) or prepaid packs. Per-user Copilot stays for human use.

Windows for developers

"Not Windows for Windows developers — Windows for developers, period." A developer-optimized OS and a secure home for local agents.

Aion 1.0 Instruct & Aion 1.0 Plan Coming months

New on-device models shipping in-box. Instruct handles summarization & rewrites; Plan is a 14B reasoning/tool-calling model with 32K context for fully local agentic work.

Microsoft Execution Containers (MXC) Early preview

A policy-driven layer declaring exactly what an agent can access (files, network), with containment enforced at runtime.

Windows Developer Configurations GA

One WinGet command sets up a distraction-free dev box — VS Code, GitHub Copilot, WSL, PowerShell 7 and tuned settings.

Intelligent Terminal Experimental

Context-aware agents brought into the terminal to debug errors and run multi-step tasks in your flow (ACP integration).

OpenClaw on Windows Preview

Runs natively and contained via MXC, with its own companion app. NVIDIA is also bringing OpenShell to Windows on MXC.

Coreutils & WSL containers GA / Preview

Linux-like CLI utilities native on Windows (GA), plus built-in Linux containers via familiar CLI & API (coming to preview).

Silicon, devices & infrastructure

Surface Laptop Ultra Later in 2026

Surface RTX Spark Dev Box Later in 2026 (US)

A local AI compute box putting serious on-device horsepower in developers' hands.

Maia 200 & Cobalt 200 New silicon

Microsoft's next-gen accelerator and CPU stack for agent-scale cloud systems, alongside a deepened NVIDIA partnership.

Project Solara Concept

A new platform exploring what comes after the laptop for always-on agents — reference designs and concept devices.

Data for the agentic era

Azure HorizonDB New

Brings PostgreSQL into agentic app development with the scale and characteristics agents need.

Fabric GPU-accelerated analytics New

A new class of GPU-accelerated analytics to make data work faster for agents and developers.

Microsoft's own models & Frontier Tuning

The MAI family steps onto center stage, with tooling to make any model fit your company.

MAI-Thinking-1 New

Microsoft's first dedicated reasoning model — built without distillation and targeted at enterprise customers.

MAI media models New

MAI-Image-2.5 & 2.5-Flash, MAI-Transcribe-1.5 and MAI-Voice-2 expand the in-house multimodal lineup.

MAI-Code-1-Flash New

A small, fast coding model — part of opening MAI model access beyond Foundry.

Frontier Tuning New

A "hill-climbing machine" that adapts models to your data. Early work includes a Mayo Clinic health frontier model and a Land O'Lakes demo.

Security

MDASH — agentic vulnerability discovery Expanded preview

A 100-agent, multi-model security scanning harness that traces full exploit chains, validates whether each flaw is genuinely exploitable, and surfaces findings inside the Microsoft Defender Portal. Scored 96.55% on the CyberGym benchmark — up ~10 points.

Microsoft Discovery GA

An agentic R&D platform whose Discovery Engine uses specialized agents to mimic the scientific method — generating hypotheses and validating theories in a continuous loop, with enterprise governance.

Majorana 2 Next-gen quantum chip

Qubits now 1,000x more reliable, with a mean lifetime of ~20 seconds (vs. microseconds). A major step toward a commercially relevant quantum machine by 2029.
